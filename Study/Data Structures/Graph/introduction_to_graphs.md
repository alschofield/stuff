**[Graphs]{.ul}**

A graph is a nonlinear data structure consisting:

-   Nodes

    -   Finite set of elements

    -   Also referred to as vertices

-   Edges

    -   Lines/arcs that connect any two nodes

    -   Finite set of ordered pairs

        -   (u,v)

[Applications]{.ul}

-   Represent networks

    -   Paths

    -   Telephones networks

    -   Circuits

-   Social media

    -   Facebook

    -   Linkedin

[Representations]{.ul}

-   Adjacency Matrix

-   Adjacency List

-   Incidence Matrix

-   Incidence List
